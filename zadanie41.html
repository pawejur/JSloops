Rodzaje pętli:

Pętla for:
konstrukcja:for (zainicjowanie_zmiennej; warunek_kończący_wykonywanie_pętli; zmiana_zmiennej) {kod który zostanie wykonany pewną ilość razy}

Pętla while z kontrolowanym wejściem:
konstrukcja:while (wyrażenie_sprawdzające_zakończenie_pętli) { ...fragment kodu który będzie powtarzany... }

Pętla do while z kontrolowanym wyjściem:
konstrukcja:Do { ...fragment kodu który będzie powtarzany... } while (false)
